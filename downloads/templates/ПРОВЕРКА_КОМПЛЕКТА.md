# Проверка стартового комплекта

Перед началом убедитесь, что присутствуют:

- [ ] `ПАСПОРТ_ОБУЧЕНИЯ.md`;
- [ ] папки `01_урок` — `06_урок`;
- [ ] по одному рабочему шаблону для уроков 1–5;
- [ ] семь файлов в `06_урок/ai-process`;
- [ ] два файла в `учебные_данные`;
- [ ] инструкция по безопасности;
- [ ] инструкция по восстановлению.

После каждого урока:

- [ ] итоговый файл сохранён в правильной папке;
- [ ] черновики не затёрли предыдущую проверяемую версию;
- [ ] `ПАСПОРТ_ОБУЧЕНИЯ.md` обновлён;
- [ ] в паспорте указаны файлы для следующего занятия.
