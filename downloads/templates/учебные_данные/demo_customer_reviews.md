# Демонстрационный набор: отзывы клиентов

Набор полностью синтетический. Он не содержит реальных имён, контактов или сведений о конкретной компании.

Продукт: условный веб-сервис для планирования задач «Фокус».

Период: одна учебная неделя.

## Что известно

- Это все отзывы, доступные для учебного примера.
- Один отзыв не равен одному уникальному пользователю: идентификация авторов удалена.
- Нельзя утверждать общую частотность проблемы по десяти отзывам.
- Нельзя устанавливать техническую причину проблемы без журналов и воспроизведения.
- Можно группировать наблюдаемые жалобы, цитировать отзывы и предлагать проверочные действия.

## Ожидаемый учебный результат

Краткая записка для руководителя продукта:

1. таблица наблюдаемых проблем;
2. подтверждающие ID и короткие цитаты;
3. уровень срочности с объяснением;
4. пробелы в данных;
5. три следующих проверочных действия.
