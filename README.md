# Сайт «ЛЁГКИЙ СТАРТ»

Статический MVP без серверной базы данных.

## Локальный запуск

Из папки `site`:

```powershell
python -m http.server 8080
```

После этого открыть:

```text
http://localhost:8080
```

Учебные тексты загружаются из Markdown-файлов, поэтому прямое открытие `index.html` через `file://` не поддерживается.

## Прогресс

Навигационный прогресс хранится в `localStorage` под ключом `light_start_progress_v1`. Учебные результаты хранятся только в файлах обучающегося.
